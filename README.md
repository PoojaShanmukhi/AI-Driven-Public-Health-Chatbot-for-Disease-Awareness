# 🩺 AI-Driven Public Health Chatbot for Disease Awareness

## 📌 Overview
HealthGuard AI is an AI-powered public health chatbot designed to provide disease awareness, symptom guidance, preventive healthcare information, and reliable responses using Google Gemini AI.

## ✨ Features
- 🤖 AI-powered chatbot using Google Gemini AI
- 🩺 Disease awareness and health education
- 💊 Symptom guidance
- 🛡️ Preventive healthcare tips
- 💬 Interactive chatbot interface
- 📍 User-friendly design

## 🛠️ Technologies Used
- React
- TypeScript
- Vite
- Google Gemini AI
- HTML
- CSS
- JavaScript

## 🚀 Installation

1. Clone the repository
```bash
git clone https://github.com/PoojaShanmukhi/AI-Driven-Public-Health-Chatbot-for-Disease-Awareness.git
```

2. Install dependencies
```bash
npm install
```

3. Add your Gemini API Key in `.env.local`

4. Start the application
```bash
npm run dev
```
